# Admin-panel

###### This project uses

  - HTML5, CSS3
  - JavaScript
  - React.js
  - Redux
  - Material Ui
  - WebPack

# Start

  Clone project from github

 ```sh
    $ git clone https://github.com/armbaldryan/sport_test
 ```

  Install the dependencies

 ```sh
    cd sport_test
    npm install
 ```
  And run the application by running webpack dev server
```sh
    cd sport_test
    npm start
 ```
